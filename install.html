Installation Instructions

Step 1: Download the latest version of Cozmo Remastered from the release page.

Step 2: Extract the contents of the downloaded zip file to a folder on your computer.

Step 3: Open the extracted folder and run the installer by double-clicking on the setup.exe file.

Step 4: Follow the on-screen instructions to complete the installation process.

Step 5: Once installed, launch Cozmo Remastered from the Start Menu or desktop shortcut.
